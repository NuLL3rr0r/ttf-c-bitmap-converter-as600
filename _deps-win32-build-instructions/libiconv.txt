1. In MSYS shell:

    mkdir _debug _release

    # debug build
    cd _debug && ../configure --prefix=/d/build/_debug-libiconv CFLAGS="-g3 -O0" CPPFLAGS="-g3 -O0" CXXFLAGS="-g3 -O0"
    make -j9 && make install

    # release build
    cd _release && ../configure --prefix=/d/build/_release-libiconv CFLAGS="-DDEBUG" CPPFLAGS="-DDEBUG" CXXFLAGS="-DDEBUG"
    make -j9 && make install

2. Copy the following:
    D:\build\release\include\* to our include dir
    D:\build\release\bin\*.dll to our rel-lib dir
    D:\build\debug\bin\*.dll to our deb-lib dir
